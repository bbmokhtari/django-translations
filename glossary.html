The terms you might see in the documentation:

supported language

A supported language is a language which is included in the LANGUAGES setting.

default language

The default language is a language which is set by the LANGUAGE_CODE setting.

translation language

A translation language is a language which is in the collection of supported languages excluding the default language.

active language

The active language is a language which is automatically activated in each request. It is usually determined by the Accept-Language header received in each HTTP request (from the browser or another client). You can access it in Django using the get_language() function.
